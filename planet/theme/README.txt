♀ Planet Venus for Plone — Theme
================================

The files in this directory comprise the theme for http://planet.plone.org/.
They're read by the `Planet Venus`_ software to generate the output HTML files
that can be served directly by a web server, and look like the snazzy Plone_
web site.

*Note*: config.ini in this directory is generated from
BASEDIR/etc/theme.cfg.in.  Please ensure that you're editing the correct file.
