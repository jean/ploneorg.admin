Books on Plone

  1. Professional Plone Development The most recent Plone book. Covers building robust, content-centric web applications with Plone 3.
  2. Content Management with Plone: Handbook for Authors and Editors End-user guide for Plone, recently updated for Plone 3.
  3. Web Component Development with Zope 3 Zope 3 development. Gives a lucid introduction to the Zope 3 technologies available to Plone.
  4. Plone Live A thorough reference to the inner workings of Plone, covers Plone 2.5.
  5. The Definitive Guide to Plone The first Plone book. Slightly outdated, but an excellent jumpstart guide for Plone.
  6. Dive into Python The de facto standard for learning Python development.
